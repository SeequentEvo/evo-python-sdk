packages.evo-blockmodels.src.evo.blockmodels.client.BlockModelAPIClient.list_versions(bm_id) async

List versions of a block model.

Returns a list of Versions for the block model referenced by bm_id, limited to the first 100 results (or the service maximum).

Versions are ordered from newest to oldest.

Parameters:

Name Type Description Default
bm_id UUID

The ID of the block model.

 required

Returns:

Type Description
list[Version]

A list of Version dataclasses for the block model, ordered newest to oldest.
